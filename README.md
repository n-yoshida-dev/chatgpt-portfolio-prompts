# chatgpt-portfolio-prompts

ChatGPT Projectsを使って開発未経験からポートフォリオを作る際に使った、学習伴走用プロンプト集です。

## この資料について

「AIに答えを出させない」「過去の文脈を踏まえたレビューをしてもらう」という方針のもと設計したプロンプト一式です。
AIをレビュアー・課題出し役として使い、自分の思考を鍛えながらポートフォリオを進めることを目的としています。

## 記事

[ChatGPT ProjectsでAIを“継続メンター化”するために作った仕組み](https://qiita.com/n-yoshida-dev/items/064adc2858ddf40c4b4f)

## ファイル一覧

| ファイル | 役割 |
|---|---|
| `00_SystemPrompt_ProjectInstructions.md` | ChatGPT Projectに直接設定するSystem Prompt |
| `01_テーマ別指導方針.md` | DB/API/コーディング/テスト/用語ごとの指導ルール辞書 |
| `02_相談テンプレート.md` | 壁打ち時の入力フォーマット。チャット冒頭と引き継ぎ時に使う |
| `03_レビュー観点チェックリスト.md` | レビュー依頼時にAIが参照する観点一覧 |
| `04_フェーズ1専用学習対象定義.md` | 自分の学習フェーズ1のスコープ・完了条件・学習順序（参考資料） |
| `05_ADRインデックス.md` | ADRの書き方テンプレートと必須候補一覧（参考資料） |
| `学習ログ.md` | 目的単位チャットが一段落したときにAIに作らせる学習ログのテンプレート |

## 注意事項

`04_フェーズ1専用学習対象定義.md` と `05_ADRインデックス.md` は私個人の学習計画に基づいた内容です。
そのまま使うのではなく、自分のロードマップに合わせて書き換えて使ってください。
